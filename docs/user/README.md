# Documentacion de usuario

Guias operativas por modulo para usuarios finales, capacitacion y soporte.

## Convencion

- Un archivo por modulo: `{modulo}.md`
- Orden obligatorio de secciones: Objetivo, Alcance, Definiciones, Responsabilidades, Desarrollo, Control de cambios
- Plantilla: [`docs/templates/USER_MODULE_DOC.md`](../templates/USER_MODULE_DOC.md)
- Generado/actualizado por el **Agente Documentador** al cerrar features (flujo multi-agente)
