# Documentacion de usuario

Guias operativas por modulo para usuarios finales, capacitacion y soporte.

Guia maestra de las tres capas (IA, desarrollador, usuario): [`docs/DOCUMENTATION.md`](../DOCUMENTATION.md).

## Convencion

- Un archivo por modulo: `{modulo}.md`
- Orden obligatorio: Objetivo, Alcance, Definiciones, Responsabilidades, Desarrollo, Control de cambios
- Plantilla: [`docs/templates/USER_MODULE_DOC.md`](../templates/USER_MODULE_DOC.md)
- Generado/actualizado por el **Agente Documentador** al cerrar features
